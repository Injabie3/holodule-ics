# Holodule-ICS

ホロライブの配信予定 [ホロジュール](https://schedule.hololive.tv/) を
任意のカレンダーアプリ等に登録できる `ics` ファイル

# 設定例

## Google Calendar (PC)

* `設定` を開く
* 左側メニュー `カレンダーの追加` -> `URL で追加`
* URL を貼り付け
* `カレンダーを追加`

PC から追加後はモバイル版 Google Calendar でも閲覧できます
